# Glassmorphism Calculator

A modern calculator with glassmorphism UI design, following the [AsmrProg tutorial](https://www.youtube.com/watch?v=sBJmRD7kNTk).

## Features

- **Glassmorphism Design**

  - Blur effects and transparent layers
  - Gradient background with subtle animation
  - Soft shadows and border highlights

- **Core Functionality**

  - Basic arithmetic operations (+, -, ×, ÷)
  - Decimal point support
  - Clear and backspace functions
  - Error handling for invalid operations

- **Technical Highlights**
  - CSS custom properties for consistent styling
  - Responsive grid layout
  - Smooth button interactions
  - Cross-browser compatibility

## Installation

1. Clone repository:

```bash
git clone https://github.com/ghtmarco/Calculator.git
```

2. Open in browser:

```bash
open index.html
```

## Usage

| Button     | Function              |
| ---------- | --------------------- |
| 0-9        | Number input          |
| .          | Decimal point         |
| +, -, ×, ÷ | Arithmetic operations |
| =          | Calculate result      |
| ←          | Backspace             |
| C          | Clear display         |

## Technologies

- HTML5 semantic markup
- CSS3 (Flexbox, Gradients, Blur Effects)
- Vanilla JavaScript (ES6)
